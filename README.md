# Draw Your Planet
A simple interactive web app where you can draw your own planet using free sketching tools, glow effects, and a clean canvas-based interface.

---

## Try it
Open the project in your browser:
[https://your-demo-link-here](https://pujitasunkara-jpg.github.io/web-os/)

---

## What it does
- Lets you freely draw on a canvas
- Create custom “planets” with your own designs
- Glow effect for colorful space-style drawings
- Clear button to reset and start again
- Smooth, simple interface built for creativity

---

## How to run it
1. Download or clone the repo
2. Open `index.html` in your browser
3. Start drawing immediately

No setup required.

---

## How it works
The app uses an HTML canvas to track mouse movement and draw strokes in real time. Each stroke is rendered with a glow effect using CSS and canvas shadow styling, giving it a space-like neon look. Clearing the canvas resets the drawing state.
